## Getting Started

First, install the dependencies:

bash
npm install

# or

yarn install

# or

pnpm install

Then, run the development server:

bash
npm run dev

# or

yarn dev

# or

pnpm dev

Open [http://localhost:5173](http://localhost:5173) in your browser to see the app.
